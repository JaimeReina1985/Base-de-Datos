Ejercicio 3: Preprocesamiento de Texto

Objetivo: Preparar los datos textuales para el entrenamiento del modelo.

Acciones:

  • Convertir todo el texto a minsculas para uniformidad.
  • Eliminar puntuacin y nmeros para reducir el ruido en los datos.
  • Eliminar palabras comunes (stop words) que no aportan significado especfico al anlisis.
  • Tokenizar el texto, es decir, dividir el texto en palabras individuales.

DOCUMENTACION

En esta etapa, preparamos los datos textuales para el entrenamiento de los modelos de PLN. El preprocesamiento es esencial para reducir el ruido y estandarizar la informacin, facilitando que los modelos aprendan patrones relevantes.

Pasos realizados:

  1. Conversin a minsculas: Se transforma todo el texto a minsculas para evitar que palabras iguales sean tratadas como diferentes por las maysculas/minsculas.
  2. Eliminacin de puntuacin y nmeros: Se eliminan todos los signos de puntuacin y los dgitos numricos, reduciendo as el ruido en los datos.
  3. Tokenizacin: Se divide el texto en palabras individuales (tokens) usando el tokenizador de NLTK.
  4. Eliminacin de stop words: Se eliminan las palabras ms comunes del espaol que no aportan significado relevante (stopwords), usando la lista de NLTK.
  5. Resultado: El texto limpio y preprocesado se almacena en una nueva columna llamada contenido_preprocesado.

Nota: Este preprocesamiento es necesario para que los modelos RNN y LSTM trabajen con datos consistentes y enfocados en el significado central de cada noticia.

import nltk
import string
from nltk.tokenize import word_tokenize
from nltk.corpus import stopwords

import keras_nlp
import tensorflow as tf

# Descargar recursos necesarios de NLTK (solo la primera vez)
nltk.download('punkt')
nltk.download('stopwords')
nltk.download('punkt_tab')

# Funcin de preprocesamiento de texto
def preprocess_text(text):
    # 1. Convertir texto a minsculas
    text = text.lower()
    # 2. Eliminar signos de puntuacin y dgitos
    text = ''.join([c for c in text if c not in string.punctuation and not c.isdigit()])
    # 3. Tokenizar el texto (dividir en palabras)
    tokens = word_tokenize(text)
    # 4. Eliminar stopwords (palabras vacas) en espaol
    tokens = [word for word in tokens if word not in stopwords.words('spanish')]
    return ' '.join(tokens)

# Aplicar el preprocesamiento a la columna 'contenido'
data_filtrada['contenido_preprocesado'] = data_filtrada['contenido'].apply(preprocess_text)

3.1 Tokenizacin y Padding

En este bloque, convertimos el texto preprocesado en secuencias numricas que puedan ser utilizadas por los modelos de deep learning.

Pasos realizados:

  1. Tokenizacin:
    Se utiliza el Tokenizer de Keras para asignar un nmero nico a cada palabra del corpus preprocesado.

  2. Conversin a secuencias:
    Cada texto se transforma en una lista de nmeros (secuencias), donde cada nmero representa una palabra.

  3. Padding:
    Las secuencias se rellenan (padding) para que todas tengan la misma longitud (max_len), facilitando su uso en modelos como RNN o LSTM.

  4. Visualizacin:
    Se muestran las primeras cinco secuencias tras el preprocesamiento y padding.

Nota: Tambin se puede experimentar con tcnicas de embedding o preprocesamientos ms avanzados, segn lo realizado en ejercicios anteriores.
